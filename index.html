		Ссылка с якорем
		Ссылки с якорем обычно используются для создания навигации внутри страницы. 
		Например, оглавление в начале страницы с большой статьей.
		Ссылка с якорем содержит символ # после которого идет идентификатор. 
		Идентификатор создается с помощью атрибута id, который может быть задан у любого тэга.
		Можно задать адрес, состоящий из одного якоря, например:
			< a href="#glava1">Глава 1
		При щелчке на такую ссылку браузер находит на странице элемент с совпадающим атрибутом id и прокручивает окно страницы к нему. 
		Т.е. перезагрузки страницы не происходит.
		Якорь можно использовать и в обычных абсолютных адресах, тогда после перехода на нужную страницу 
		будет происходить прокрутка к заданной части этой страницы.
